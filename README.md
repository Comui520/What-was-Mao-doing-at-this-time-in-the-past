# 此时此刻，毛泽东在做什么？

一个展示历史人物时间线的互动网站。

## 项目说明

根据当前的时间和日期，展示毛泽东在1945-1949年解放战争时期此时此刻正在做什么。

- 如果当前日期和时间对应历史上的重大事件，则显示该历史事件
- 否则显示毛泽东在这个时间点的日常活动

## 技术栈

- Vue 3
- Vite
- 原生 CSS

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 数据文件

所有历史数据存储在 `data/` 目录下：

- `daily_routine.json` - 日常作息时间表
- `historical_events.json` - 重要历史事件

## 添加更多历史事件

编辑 `data/historical_events.json`，按照以下格式添加：

```json
{
  "date": "1948-09-12",
    "time": "10",
    "event": "发起辽沈战役",
    "description": "1948年9月12日上午，毛泽东在西柏坡指挥发起辽沈战役。这是解放战争中三大战役的第一场，目标是解放东北全境。毛泽东亲自起草了多份电报，部署林彪率领的东北野战军作战方针。此战历时52天，歼灭国民党军47万余人。",
    "mood": "沉着、果断、运筹帷幄",
    "impact": "高",
    "historical_context": "1948年秋，解放战争进入战略决战阶段。国民党军队总兵力下降到365万人，解放军总兵力上升到280万人，力量对比发生根本变化。"
}
```
