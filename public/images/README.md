# 图片说明文档

本项目需要以下图片文件，请将图片放置在此目录下。

## 1. 毛泽东人像（顶部红条左侧）

**文件名：** `mao-portrait.png`

**位置：** `public/images/mao-portrait.png`

**要求：**
- **尺寸：** 建议 400x400 像素或更大（正方形或竖向）
- **格式：** PNG（推荐，支持透明背景）或 JPG
- **内容：** 毛泽东肖像照片
- **背景：** **红色背景**（重要！）与顶部红条相呼应
- **建议：**
  - 使用红底的毛泽东肖像照片（如经典的红底证件照风格）
  - 人物居中或稍偏左
  - 面部清晰，表情庄重
  - 推荐使用 1945-1949 年期间的照片，符合网站主题
  - 如果是 PNG 格式，可以抠图去掉背景，让红色背景自然融入

**显示效果：**
- 显示在顶部红条左侧
- 尺寸约 200x200px
- 不会被裁剪成圆形，保持原始形状
- 红底图片会与红色背景自然融合
- 鼠标悬停时会轻微放大
- 有阴影效果增强立体感

**推荐图片类型：**
1. 红底证件照风格的毛泽东肖像
2. 或者使用 PNG 透明背景的毛泽东肖像（背景会自动显示为红色）

---

## 2. 毛泽东诗词背景（主内容区背景）

**文件名：** `mao-poetry-bg.png`

**位置：** `public/images/mao-poetry-bg.png`

**要求：**
- **尺寸：** 建议 1200x800 像素或更大
- **格式：** PNG（推荐）或 JPG
- **内容：** 毛泽东手书诗词或题字
- **建议：**
  - 使用毛泽东的草书作品（如《沁园春·雪》、《七律·长征》等）
  - 背景为宣纸或古朴纸张质感
  - 墨迹清晰，有艺术感
  - 推荐使用横向构图的诗词作品

**显示效果：**
- 作为主内容区的背景图片
- 透明度设置为 8%（非常淡，不影响阅读）
- 铺满整个内容区域
- 不会影响前景文字的可读性

---

## 图片来源建议

1. **公开资源：**
   - 维基百科（公有领域图片）
   - 中国国家博物馆官网
   - 新华社图片库

2. **注意事项：**
   - 确保图片使用权限
   - 优先选择公有领域或允许非商业使用的图片
   - 图片质量要清晰，避免模糊或像素化

---

## 临时占位图片

如果暂时没有合适的图片，可以：
1. 使用纯色占位图
2. 或者暂时注释掉图片相关的 CSS 代码
3. 网站功能不受影响，只是视觉效果会简化

---

## 安装步骤

1. 将图片文件重命名为指定的文件名
2. 放置到 `public/images/` 目录下
3. 刷新浏览器即可看到效果

如果图片不显示，请检查：
- 文件名是否正确（区分大小写）
- 文件路径是否正确
- 浏览器控制台是否有 404 错误
